📊 總覽頁
  • • 加權指數 + 那斯達克即時指數
  • • 市場溫度計（0–100°，由RSI+漲跌比計算）
  • • 族群表現橫條圖（面板/記憶體/載板等）
  • • 情緒分布：強漲/小漲/持平/小跌/強跌
  • • 排行榜：漲跌幅 / 成交量 / 綜合評分
📈 個股卡片
  • • 右上角圓形數字 = 綜合評分（0–100），分數越高代表技術面越強
  • • 評分 ≥ 70 顯示高亮色，45–69 金色，< 45 偏空色
  • • 台股：漲紅跌綠｜美股：漲綠跌紅
  • • 訊號徽章：🟢進場 / 🔴減碼 / 🟡觀察
  • • 點擊卡片開啟個股詳細分析頁
  • • 走勢圖疊加 MA5（黃）/ MA20（紫）
✦ AI 選股助理
  • • 6 個分析情境（短線反彈/中期布局等）
  • • 本地篩選引擎即時計算，不消耗額度
  • • Groq AI 生成三欄報告（推薦/觀察/風險）
  • • 推薦結果自動記錄到「追蹤記錄」
  • • 追蹤記錄即時顯示損益百分比
🔍 個股詳細頁

  • • 籌碼面：三大法人買賣超 + 融資融券
  • • 成交量分析：量比 + 近10日量能圖
  • • 按鈕觸發：操作建議 / 新聞 / AI分析
  • • 手機版底部抽屜，向下滑動關閉

🎯 訊號邏輯（需三條件同時成立）
🟢 進場訊號
強訊號（各+2）：
· RSI 在 28–58 且比前日上升
· MACD 柱狀剛從負翻正
輔助訊號（各+1）：
· 均線多頭排列（MA5 > MA20）
· MACD 柱狀偏多（> 0）
· 近5日上漲
→ 總分 ≥ 3 且含至少一個強訊號
🔴 減碼警示
強訊號（各+2）：
· RSI ≥ 68 且比前日下降
· MACD 柱狀剛從正轉負
輔助訊號（各+1）：
· 均線空頭排列（MA5 < MA20）
· MA5 開始下彎
· 近5日下跌超過 1%
→ 總分 ≥ 3 且含至少一個強訊號
🟡 持續觀望
· 進場或減碼分數達 1–2 分
· 或 MACD 即將黃金交叉
· 訊號尚未明確，等待確認
→ 持續追蹤，不急於行動
