STOCK RADAR 使用說明
台股 × 美股個股監控儀表板 · 資料僅供參考，不構成投資建議
📊 總覽頁
  • • 加權指數 + 那斯達克即時指數（含 30 日 sparkline）
  • • 市場溫度計（0–100°，由 RSI + 漲跌比計算）
  • • 族群表現橫條圖 — 點擊直接切換到該族群
  • • 情緒分布：強漲 / 小漲 / 持平 / 小跌 / 強跌
  • • 排行榜：漲跌幅 / 成交量 / 綜合評分（可切換全部/台股/美股）
📈 個股卡片
  • • 即時報價 + 漲跌幅（台股 TWSE MIS，備援 Yahoo Finance）
  • • 右上角綜合評分 0–100，歷史資料載入前顯示「—」
  • • 台股：漲綠跌紅 / vs 大盤：個股漲跌幅超越大盤幅度
  • • 訊號徽章：● 進場 ● 減碼 / ● 觀察（hover 查看觸發條件）
  • • ☆ 按鈕加入自選清單，點擊卡片開啟個股詳細頁
🔍 全域搜尋 更新
  • • / 開啟全域搜尋 / Esc 關閉
  • • 輸入代號、名稱或快捷鍵開啟
  • • ↑↓ 選擇結果 · Enter 開啟個股詳細頁
  • • 搜尋結果即時顯示現價和漲跌幅
  • • 全市場 1800+ 支：搜尋結果顯示「全市場」區塊可直接加入監控
⭐ 自選清單 更新
  • • 股票卡片右下方 ☆ 按鈕加入 / 移除自選
  • • 股票 Tab「自選」可查看所有自選股
  • • 自選頁頂部搜尋框可快速搜尋並加入
  • • ✕ 卡片移除個股，可透過搜尋重新加入
  • • 登入後資料儲存於 Firestore，多裝置同步
📉 個股詳細頁 — 走勢面
  • • 右上切換 折線 / K線 圖表類型
  • • K 線：hover 顯示 OHLC（開高低收 + 成交量）tooltip
  • • 時間框架：1M / 3M / 6M / 1Y（台股 6M/1Y 較慢）
  • • 疊加 MA5（黃）/ MA20（紫）均線
  • • 台股：漲紅跌綠 / 美股：漲綠跌紅
📊 個股詳細頁 — 分析面板
  • • 技術指標：RSI · MACD · 均線排列 · 布林通道
  • • 籌碼面：三大法人 · 融資融券（含近 5 日趨勢圖）
  • • 基本面：P/E · 股息 P/E · 殖利率 · EPS · 股價淨值比 · 市值
  • • vs 大盤：相對大盤強度顯示於各面板左上角
  • • 操作建議 / 新聞 / AI 分析（手動觸發）
✦ AI 選股助理
  • • 情境（上排）控制 AI 分析框架和排序，不影響篩選
  • • 篩選條件（下排）獨立控制顯示流量，互不影響
  • • 篩選條件：技術面 / 量能 / 法人動向 / 熱群
  • • 分析結果：強烈推薦 / 值得關注 / 注意風險 三欄
  • • 推薦結果自動記到追蹤記錄，顯示即時勝率和勝率
📋 選股追蹤記錄 更新
  • • AI 推薦後自動記錄進場價格和推薦日期
  • • 即時計算每筆推薦的報酬百分比
  • • 主畫面顯示最近 5 筆 + 統計（勝率 / 平均報酬）
  • • 點 Header 📋 追蹤記錄 按鈕查看完整清單
  • • 進場價可手動修改（✏️ 欄位直接輸入），同步雲端
  • • 最多保留 30 筆，儲存於 localStorage + Firestore

● 進場訊號
強烈訊號（各+2）：
· RSI 在 28–38 且比較前日上升
· MACD 且MACD死叉後止跌

輔助訊號（各+1）：
· 均線多頭（MA5 > MA20）
· MACD 金叉（> 0）
· 前5日上漲
→ 總分 ≥ 3 顯示強烈訊號
→ 連續 2 次連續顯示
● 減碼警示
強烈訊號（各+2）：
· RSI ≥ 68 且比較前日下降
· MACD 死叉後持續下降跌值

輔助訊號（各+1）：
· 均線空頭（MA5 < MA20）
· MACD 死叉（< 0）
· 前5日下跌 > 1%
→ 總分 ≥ 3 顯示強烈訊號
→ 連續 2 次連續顯示
● 持續觀望
· 確認減碼訊號 1–2 分
· 確 MACD 即將金交叉
· 即將確認 2 個條件

→ 繼續或觀察 2 次確認
→ 持續觀察，先行動

⚠️ 重要注意事項
  • • 本平台資料僅供參考，不構成任何投資建議，買賣請自行判斷。
  • • 即時行情來自 TWSE MIS API + Yahoo Finance，盤中可能有 15 分鐘延遲。
  • • 三大法人 / 融資融券資料為收盤後更新，盤中只顯示一日趨勢。
  • • 基本面資料（P/E、殖利率等）來自 Yahoo Finance，可能有延遲或缺漏。
  • • AI 分析由 Groq（llama-3.3-70b）生成，其間摘要由 Gemini 2.5 Flash 提供。
  • • Groq / Gemini 為免費額度，AI 功能採手動觸發，避免額度耗盡。
  • • 評分與訊號需要歷史資料載入後才能計算，載入前顯示「—」為正常。
  • • 籌碼追蹤記錄由自選清單儲存於 localStorage + Firestore，清除快取後消失。
  • • K 線 6M / 1Y 台股需多次 API 請求，載入後較慢屬正常。
